# Tutorial Screenshots

This folder contains tutorial screenshots for testing.

## Usage
1. Drop your presentation screenshots here
2. Name them sequentially: , , etc.
3. Any format works (PNG, JPG) - will be auto-converted to WebP

## Sizing Guidelines
- Screenshots will be automatically resized to max 1200px width
- Files will be compressed to ~200KB each
- Original aspect ratio preserved

## Testing
After uploading screenshots here, we'll test the upload feature locally before deploying to Supabase Storage.
